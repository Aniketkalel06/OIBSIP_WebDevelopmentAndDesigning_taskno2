# Personal Portfolio Website

## About This Project

This project is a Personal Portfolio Website developed as part of my Web Development & Designing Internship under OIBSIP at Oasis Infobyte.

The purpose of this project is to create a responsive and visually appealing portfolio website to showcase my skills, projects, and contact details.

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## Project Description

This portfolio website contains multiple sections such as:

- Home section with introduction
- About section describing my background
- Skills section highlighting my technical skills
- Projects section showcasing my work
- Contact section for communication

The website is designed to be simple, clean, and user-friendly.

---

## How It Works

HTML is used to structure the layout of the portfolio.
CSS is used to style the website and make it responsive.
JavaScript is used to add interactivity and smooth user experience wherever required.

The design is responsive so that it adjusts properly on different screen sizes.

---

## How to Run the Project

1. Download or clone this repository.
2. Open the `index.html` file in any web browser.
3. Navigate through the different sections of the portfolio.
